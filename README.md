## Installation üöÄ

To install this project, follow these steps:

1. Clone the repository:

```bash
git clone https://github.com/p55d2k/project-taskcraft-frontend.git
```

2. Install the dependencies:

```bash
cd project-taskcraft-frontend
bun install
```

3. Run the project:

```bash
bun dev
```
